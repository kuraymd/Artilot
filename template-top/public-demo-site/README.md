# Public Demo Site

販売用テンプレート本体を公開せずに、雰囲気だけを見せるための公開デモカタログです。

## 方針

- このフォルダには販売用テンプレートのHTML/CSS/JSをコピーしません。
- カードはカテゴリ、テーマ、用途、色の雰囲気だけを表示します。
- 将来スクリーンショットを載せる場合は、`images/previews` などに画像だけを追加してください。

## GitHub Pagesで使う場合

このフォルダの中身を `website-template-demo` リポジトリの公開用ルートに置くと、
`https://kuraymd.github.io/website-template-demo/` で一覧ページとして表示できます。
